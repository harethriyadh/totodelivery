## 1. Authentication

**Method**: `POST`
**URL**: `/auth/login`
**Body**:
```json
{
  "username": "market1", // or "driver1"
  "password": "0000",   // or "1234"
  "platform": "MARKET_DASHBOARD", // or "DRIVER_APP"

```
**Expected Response**: `{ "success": true, "token": "JWT...", "user": { "id": "...", "role": "market_owner" } }`

### Refresh Token
**Method**: `POST`
**URL**: `/auth/refresh-token`
**Body**: `{ "refresh_token": "..." }`

## 3. Products (Market)

### Get Products
**Method**: `GET`
**URLs**: 
- `/products/market` (Get all market products)
- `/products/category/:id` (Get by category)

### Create Product
**Method**: `POST`
**URL**: `/products`
**Expected Body Mapping**:
```json
{
  "name": "Fresh Oranges",
  "category": { "id": "fresh_veg", "label_ar": "فواكه", "parent_id": "root" },
  "pricing": { "price": 2500, "cost_price": 2000 },
  "inventory": { "stock_quantity": 100, "unit_type": "kg" },
  "attributes": [{ "key": "origin", "value": "Local", "label": "Origin" }],
  "brand": "TOTO Farm",
  "tags": ["fresh", "organic"]
}
```

### Toggle Active
**Method**: `PATCH`
**URL**: `/products/<PRODUCT_ID>/toggle-active`
